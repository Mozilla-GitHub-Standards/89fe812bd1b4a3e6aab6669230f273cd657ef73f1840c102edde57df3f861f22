around
======

HTML5 foursquare client.

## Installation ##

You need `gem` and `npm` (and thus Ruby and Node.js).

    gem install sass
    gem install --version '~> 0.9' rb-fsevent
    npm install
    ./serve

Then head to [localhost:8008](http://localhost:8008/).
